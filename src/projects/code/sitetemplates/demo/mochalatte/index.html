Heading 1

This website template was my entry for The 32-Bit Cafe Spring 2025 Tea Party. Therefore, this template is open for public use.

I am a coffee lover, so I participated in the event, and designed and named this tempalte after my favourite coffee flavour, mocha latte.

Feel free to study the HTML and CSS of this template, use the code and customise it yourself!

Features

Main features of this layout's design include:

  • Light and dark mode support, by respecting user's system settings
  • Mobile-friendliness
  • Accessibility:
    • Colour scheme with good colour contrasts
    • Friendly to keyboard navigation, with visible outlines for when interactive elements (including links, buttons and form inputs) are focused on, as well as a "skip to content" link (visible only when you press the Tab key when you are at the top of the page) and a "back to top" link (at the footer)
