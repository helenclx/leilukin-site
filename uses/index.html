Below is a non-exhaustive list of the hardware and software that I use.

You can find more /uses pages on uses.tech.

Hardware

#
  • Laptop: Acer Predator Helios 300. Purchased in February 2018.
  • Smartphone: Xiaomi Redmi Note 13 Pro 5G. Purchased in August 2024.
  • Smartwatch: Xiaomi Smart Band 9

Software

#

General

#
  • Fonts: Lexend, Readex Pro, and Intel One Mono

Operating Systems

#
  • Laptop: Windows 11 Pro, with Windows Subsystem for Linux (WSL) enabled and Ubuntu installed as the Linux distribution
  • Smartphone: Xiaomi HyperOS 2.0, based on Android 15
  • Smartwatch: Xiaomi HyperOS 2.0

Operating System Utilities

#
  • Chris Titus Tech's Windows Utility
  • WinScript
  • MIUI-Debloater
  • Package manager: Windows Package Manager (winget) and APT on Linux
  • Mobile app manager: Obtanium
  • Screen reader: NVDA on Windows, and TalkBack on Android

Internet

#
  • Internet browser: Mozilla Firefox, Vivaldi, Lynx, and Tor Browser
  • Search engine: Ecosia and Mullvad Leta
  • Email service: Purelymail, Proton Mail and Gmail
  • Email client: Mozilla Thunderbird
  • Feed reader: FreshRSS (32-Bit Cafe instance), and Readrops on Android
  • VPN: Proton VPN
  • FTP client: WinSCP
  • IRC client: IRCCloud
  • Mastodon client: The official web interface on desktop, and Moshidon on Android
  • YouTube frontend: FreeTube on Windows, and NewPipe on Android
  • Reddit frontend: Redlib

Productivity

#
  • Notes:
    • Obsidian for writing, project management and personal knowledge base
    • Standard Notes for quick notes
  • Cloud file storage:
    • pCloud (Lifetime paid plan)
    • Tresorit for storing my most important files
    • MEGA for storing personal photos
    • Nextcloud (The Good Cloud instance)
  • Bookmarks: Mozilla Firefox bookmarks and Raindrop.io
  • Text editor: Notepads
  • Office suite: LibreOffice
  • Calendar: Nextcloud (The Good Cloud instance) with Mozilla Thunderbird
  • Tasks: Nextcloud (The Good Cloud instance) with Mozilla Thunderbird
  • Personal finance: Money Manager EX
  • Calculator: Qalculate!
  • Spellchecker: LanguageTool
  • File archiver: PeaZip

Security

#
  • Password manager: BitWarden and KeePassXC
  • Multi-factor authentication Tool: Ente Auth
  • Cloud file storage encryption: Cryptomator

Software Development

#
  • Integrated development environment (IDE): Neovim with the LazyVim setup
  • Code editors: Vim and Notepad++
  • Terminal emulator: Windows Terminal
  • Shell: PowerShell on Windows and Bash on Linux
  • Version control: Git with GitHub

Media

#
  • Image viewer: ImageGlass on Windows, and Simple Gallery on Android
  • Videos: MPC-BE on Windows, and mpv on Android
  • Audio: MPC-BE on Windows, and mpv on Android
  • Music library: Dopamine on Windows, and Simple Music Player on Android
  • Ebooks: SumatraPDF for Windows and Librera Reader FDroid for Android
  • PDF reader: SumatraPDF on Windows, and MJ PDF on Android
  • Comic reader: YACReader
  • Screenshot tool: ShareX
  • Screen recorder: Bandicam (also used for taking video game screenshots)
  • Image converter: ImageMagick
  • Media converter: Shutter Encoder and FFmpeg
  • Game streaming: Sunshine and Moonlight

Art and Design

#
  • Photo editor: Affinity Photo 2
  • Vector graphic editor: Affinity Designer 2
  • Drawing: Krita
  • Video editor: Kdenlive
  • Audio editor: Tenacity
